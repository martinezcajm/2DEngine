static Label* Label::CreateLabel ( )

Factory that creates labels.

Checks that the number of labels didn't pass the maxim amount established If you wish to create a Label you must use this method. In case the maximum amount of labels has been reached it will return nullptr. Otherwise it will return a pointer to a label.

Returns
Label* returns the label created or nullptr if the maximum of labels has been reached
